# Лабораторная работа №1 — Герои, Злодеи, Монстры (вариант 5)

Реализованы классы:
- `Character` — абстрактный базовый класс
- `Hero` — герой с именем, оружием и навыками
- `Villain` — злодей с именем, оружием, злодеянием, местом обитания и навыками
- `Monster` — монстр с именем и описанием внешности
- `Keeper` — контейнер для хранения, редактирования, сохранения и загрузки
- `CustomExceptions` — исключения, ошибки

## Сборка и запуск

Сборка:
  g++ *.cpp -o lab1

Запуск:
  ./lab1

Проект соответствует требованиям методички ГУАП.
